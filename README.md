# 📄 AI Resume Builder

A professional resume builder web application built using **Streamlit**.

## 🚀 Live Demo

🔗 https://harish-resume.streamlit.app

---

## ✨ Features

* Create professional resumes easily
* Add profile summary, education, skills, projects
* Include soft skills and languages
* Upload profile image/logo
* Clickable GitHub & LinkedIn links
* Download resume

---

## 🛠️ Tech Stack

* Python
* Streamlit
